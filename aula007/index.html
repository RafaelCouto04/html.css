Prinpipais formatações

Negrito / Destaque

Você pode destacar em negrito utilizando a tag "b"

Ou pode destacar a frase utilizando "strong"

Italico / Ênfase

você pode destacar a palavra em italico usando a tag "i"

Ou você pode dar ênfase em uma frase utilizando a tag "em"

Texto Marcado

Podemos criar um texto marcado usando a tag "mark"

Esse exemplo é somente para evidenciar a outra opção de o style de mark

Texto grande e pequeno

Estamos criando um texto grande utilizando a tag "big" e um texto pequeno utilizando a tag "small" nesse paragrafo

Texto Deletado

Podemos marcar um texto como excluido para evidenciar que ele pode ser lido mas deve ser desconsiderado utilizando a tag "del"

Texto Inserido

Podemos marcar um texto como inserido para dar enfâse e indicar que foi adcionado depois utilizando a tag "ins"

Também podemos sublinhar o texto usando a tag "u"

Texto Sobrescrito

Para inserir textos sobrescritos você deve usar a tag "sup", como no exemplo: x2+3

Texto subscrito

Para inserir textos sobrescritos você deve usar a tag "sub", como no exemplo:H2O
